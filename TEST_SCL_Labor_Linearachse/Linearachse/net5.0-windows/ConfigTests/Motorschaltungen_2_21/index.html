Motorschaltungen 2 - 2.1 (Linearachse)


→ Linderonline/FK

Beschreibung

Ein Drehstromasynchronmotor soll direkt geschaltet werden. Der Zyklus kann nicht abgebrochen werden.
  • Sobald der Taster ① gedrückt wird, leuchtet die grüne Kontrollleuchte
  • Der Motor läuft 2 Sekunden lang im RL
  • Danach reversiert er 2 Sekunden lang
  • Der Zyklus ist damit abgeschlossen und die grüne Kontrollleuchte erlischt


Es werden keinerlei Störungen ausgewertet.

DigInput.S1 -S1 Taster Start ( ① )
DigOutput.P4 -P4 Meldeleuchte grün
DigOutput.Q1 -Q1 Linearachse Rechtslauf
DigOutput.Q2 -Q2 Linearachse Linkslauf
